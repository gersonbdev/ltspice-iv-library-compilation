This SPICE library contains all models for SMD 
inductances of EPCOS SIMID series from EIA sizes
0402 to 2220.

You will get good results regarding the imaginary
part of the impedance for all parts. For some parts
the accuracy of the real part (e.g. Q-factor) is 
reduced due to limitations in the model.

For most applications the accuracy will be sufficient.

We are already working to improve our models and try
to make them available for our customers as soon as
possible.

Installation of SIMID Model Library	          Date: 1.12.98
===============================================================

The model library of SIMID (SMD inductors) for the 
circuit design program PSpice, version 6.1 and following, 
comprises the following two files:

SIMID.LIB	ASCII data file:
	        This file respresents the actual library and 
	        contains the data required for calculation. 

SIMID.SLB	Symbol data file:
	        This file contains the terminal/graphics information
	        required by the graphic user interface "Schematics"

To install the library you must copy the a.m. files into 
the "PSpice working directory"\LIB .

The library files must then be activated under PSpice. 
Proceed as follows to install the SIMID library :

1.	Copy the files SIMID.lib and SIMID.slb into
	PATH /LIB or /UserLib

2.	Start PSpice
	Install library file SIMID.LIB as permanent library
	("ADD LIBRARY*") in submenu (version 6.1) 
	ANALYSIS -> LIBRARY AND INCLUDE FILES ...
	BROWSE.. ( CHOOSE SIMID.LIB ) 
		.. ADD* for general include of library
		.. OK 

	ATTENTION ! Add without asterisk ("*") is a local INCLUDE
		only for this actual simulation task.

3.	Install symbol data file SIMID.SLB for the
	"Schematics" graphic interface in submenu
	OPTIONS -> EDITOR CONFIGURATION -> LIBRARY SETTINGS
	BROWSE.. ( CHOOSE SIMID.LIB ) .. ADD* .. OK
